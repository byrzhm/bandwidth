|-------|------------|-----------------|-------------|

| DDR3  |            |                 | ~12.8 GB/s  |
| DDR4  |            |                 | 19.2 GB/s   |
| HBM1  |            |                 | 128 GB/s    |
| HBM2  |            |                 | 307 GB/s    |
| HBM2E |            |                 | 461 GB/s    |
| HBM3  |            |                 | 819 GB/s    |
